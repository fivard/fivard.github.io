Detailed Description

Main class, which contains message.

Each message have unique id

Example

MessageLog log("data");
MessageLog
Main class, which contains message.
Definition: MessageLog.h:110
Note
There are two vectors: log and temp. Log is the main vector which contains messages which is created by user.
Temp is used only in some functions.
